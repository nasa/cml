## Synopsis

This model provides an integrable-object comprising a 3-vector.
An integrable-object is a single entity that can be added to a simulation's
integration loop so that it may be integrated as a single entity along with all
other integrable variables.

E.g. intgegrating a force to obtain accumulated impulse.
