## Synopsis

Provides the ability to set the core-properties of a mass body based on known
composite-properties. Typically, the assignment goes the other way, with the
composite-properties being derived from the combination of core-properties.
This model allows core properties at some point in the mass-tree to be
manipulated to generate a specified set of composite-properties.
