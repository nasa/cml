## Synopsis

This model provides the capbility to manage the mass and mass distribution of
a body with a variable mass, such as a fuel tank.  It provides interpolation
capabilities for the center-of-mass position and inertia tensor.

Dynamic in this sense refers to the changing mass.  It does not model the
dynamic effects of internal mass-motion.
