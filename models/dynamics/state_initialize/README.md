## Synopsis
Provides a set of models that are primarily purposed towards populating
the JEOD state-initialize BodyAction model.
The models included in this set translate multiple input options into the
format required by JEOD for initialization of a vehicular state including
position, velocity, attitude, and attitude rate.

## Documentation

These models are mostly independently documented in their respective
docs directories.

## Verification

These models mostly contain independent verification test cases in their
respective verif directories.
