## Synopsis

This model provides a front-end to the JEOD state-initialize BodyAction model.
It translates multiple input options into the format required by JEOD for
initialization of a vehicular state including position, velocity, attitude,
and attitude rate.
Provides multiple options for initializing a vehicle state.
