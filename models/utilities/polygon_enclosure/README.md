## Synopsis

Provides a mechanism for determining whether a point is enclosed within a
prescribed convex polygon. The model has separate implementations for a polygon
on a flat Euclidean plane, and for a polygon on a spherical surface.

## Documentation

This model is independently documented in the docs directory; the documentation
is incomplete.

## Verification

This model contains independent verification test cases in the verif directory;
the verification is preliminary and needs additional test cases.
