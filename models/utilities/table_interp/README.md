## Synopsis

A legacy table-interpolation code-base. This model functions well for
isolated and relatively simple linear interpolation activities and is easy to
use.

Where interpolation activities are being conducted as part of routine updates,
the C++ version (table\_interp\_cpp) is more computationally efficient.

Where interpolation activities are more demanding than simple linear
interpolation between a few data points, the extensibility inherent in the
C++ version (table\_interp\_cpp) provides more capabilities.
