## Synopsis

A fault-injection architecture. The former fault-arch model has been replaced
with an updated architecture, available in fault\_management.

The old contents of this model have been moved to fault\_arch\_legacy

The contents in this directory have been replaced with an interface to the
updated fault\_management model.

This model is intended to provide a temporary bridge to legacy models looking
for content at fault\_arch. If this model fails for any reason, the correct
solution is to update the legacy model to use fault\_management instead of the
deprecated fault\_arch.
